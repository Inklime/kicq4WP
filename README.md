# kicq4WP

**kicq4WP** — это нативный ICQ-клиент для Windows Phone 8.1+, реализующий поддержку протокола OSCAR.

## ⚙️ Описание

Клиент написан на C# с использованием WinRT (Windows Runtime API). Он поддерживает прямое подключение к серверу ICQ через OSCAR и реализует базовые функции обмена сообщениями и работы со списком контактов.

Проект вдохновлён дизайном Windows Phone и работой существующих ICQ-клиентов, таких как QIP и Jimm.

---

## ✅ Реализовано:

- 📡 Подключение к серверу `195.66.114.37:5190` через `StreamSocket`
- 🔐 Авторизация через DirectAuth (MD5-хеш 9 нулей + ASCII UIN в TLV 0x4C)
- 📋 Отправка TLV-пакетов с поддержкой:
  - UIN (0x01)
  - MD5-хеш (0x4C)
  - Client ID (0x03)
  - Версия, билд, язык, страна (0x16 – 0x1D)
  - Прочие служебные (0x14, 0x5A)
- 🧾 Логирование всех этапов подключения и авторизации (Debug.WriteLine)
- 🌐 Получение списка контактов (заглушка с примерами)
- ✉️ Отправка сообщений (заглушка, форматирование подготовлено)
- 🎨 Интерфейс в стиле Windows Phone
- 📶 Выпадающий список статусов (`В сети`, `Отошёл`, `Невидимка` и др.)

## 📄 Протокол

Клиент использует протокол **OSCAR** (ICQ), следуя спецификации:
- FLAP-кадры
- SNAC-заголовки
- TLV-пары в авторизации
- Прямая авторизация без запроса challenge

Частично реализовано поведение QIP при анализе трафика через Wireshark.
