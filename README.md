# kicq4WP

**kicq4WP** — это нативный ICQ-клиент для Windows Phone 8.1+, реализующий поддержку протокола OSCAR.

## ⚙️ Описание

Клиент написан на C# с использованием WinRT (Windows Runtime API). Он поддерживает прямое подключение к серверу KICQ через OSCAR и реализует базовые функции обмена сообщениями и работы со списком контактов.

---

## ✅ Реализовано:

- 📡 Подключение к серверу `195.66.114.37:5190` через `StreamSocket`
- 🔐 Авторизация через DirectAuth (Plain XOR)
- 🧾 Логирование всех этапов подключения и авторизации (Debug.WriteLine)
- ✉️ Отправка сообщений
- 📶 Выпадающий список статусов (`В сети`, `Отошёл`, `Невидимка` и др.)
- 👥 Полноценная загрузка списка контактов с сервера
- 👤 Отображение статусов контактов в списке c текущими статусами (статусы иногда не отображаются)
- 👥 Сохранение контактов
- 📥 Приём входящих сообщений
- 💬 Отображение истории чата
- ↔️ Изменение статуса в реальном времени
- 🔔 Уведомления о входящих сообщениях
- 🔌 Переподключение при потере соединения
- 👥 Изменение списка контактов


---

## 🚧 В процессе реализации:


- 🔐 Логин с MD5
- 📤 Обработка SNAC-сообщений и ошибок от сервера


---

## 📄 Протокол

Клиент использует протокол **OSCAR** (ICQ), следуя спецификации:
- FLAP-кадры
- SNAC-заголовки
- TLV-пары в авторизации
- Прямая авторизация без запроса challenge

Частично реализовано поведение QIP при анализе трафика через Wireshark.
